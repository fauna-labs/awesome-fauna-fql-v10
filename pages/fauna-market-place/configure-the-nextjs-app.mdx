# Configure the Next.js app

The sample application is written in Next.js. Create a new `.env` file in the root of your application and the following environment variable.

```jsx
NEXT_PUBLIC_FAUNA_ENDPOINT="https://db.fauna.com/"
NEXT_PUBLIC_FAUNA_SECRET="<Your Fauna Secret>"
```

Run the app by running the following commands in your terminal

```bash
$ npm i
$ npm run dev
```

Visit http://localhost:3000
